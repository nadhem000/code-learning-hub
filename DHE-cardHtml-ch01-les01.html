What is HTML?

Chapter 1: Getting Started with HTML
Chapter progress 0/4
Course progress 0/25

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and meaning of web content.

A Brief History of HTML

HTML was first developed by Tim Berners-Lee in 1991 while working at CERN. The initial version consisted of a small set of tags. In 1995, HTML 2.0 standardised many features. The browser wars of the late 90s brought HTML 4.01 (1999) which became the dominant language for web pages. The current version is HTML5, finalised in 2014, which introduced powerful new elements like <video>, <canvas>, and semantic tags (<header>, <footer>, <article>), as well as APIs for complex web applications.

What is HTML used for?

HTML is the backbone of every website. It structures content into headings, paragraphs, lists, links, images, tables, forms, and more. Browsers read HTML documents and render them as visual web pages. HTML also provides semantic meaning (e.g., <nav> for navigation, <aside> for side content), which helps search engines and assistive technologies understand the page. Modern HTML5 is used not only for traditional websites but also for web applications, email templates, and even cross-platform mobile apps (via frameworks).

Basic Requirements to Write HTML

You don't need any special software to start writing HTML. A simple text editor (like Notepad, VS Code, or Sublime Text) and a web browser (Chrome, Firefox, Edge) are enough. Save your file with a .html extension and open it in a browser to see the result. No internet connection is required; HTML files run locally. As you progress, you might use code editors with syntax highlighting, live preview, and developer tools for debugging.

How HTML Works in a Browser

When you open an HTML file in a browser, the browser's rendering engine parses the HTML code and constructs a Document Object Model (DOM) tree. It then applies any CSS styles and executes JavaScript to create the interactive page you see. The HTML markup tells the browser what each piece of content is (a heading, a paragraph, an image), so it can be displayed appropriately.

A Simple HTML Document Explained

A minimal HTML document looks like this:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

Core HTML Concepts

Key elements: <!DOCTYPE html> declares the document type, <html> is the root, <head> contains meta information, and <body> holds the visible content. Elements are usually composed of an opening tag, content, and a closing tag (e.g., <p>...</p>). Some elements are self-closing, like <img> and <br>. Attributes provide additional info (e.g., href, src, alt).

HTML, CSS, and JavaScript: The Trio of Web Development

HTML alone gives structure, but to make pages visually appealing you use CSS (Cascading Style Sheets) for styling (colors, fonts, layout). JavaScript adds interactivity (animations, form validation, dynamic content). In this course, you'll learn how these three technologies work together to create modern websites.

HTML is the foundation of every website. Combined with CSS and JavaScript, it creates rich, interactive experiences. By mastering HTML, you take the first essential step toward becoming a web developer. In the next lessons, we'll dive deeper into elements, attributes, and building your first real web page.
